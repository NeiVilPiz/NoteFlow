# Gestión del Proyecto

## Metodología

El desarrollo de NoteFlow se gestiona mediante un tablero Kanban en Trello.

Las columnas utilizadas son:

### Backlog

Ideas y tareas futuras pendientes de planificación.

### Todo

Tareas listas para comenzar.

### In Progress

Tareas que están siendo desarrolladas actualmente.

### Review

Funcionalidades implementadas pendientes de revisión o pruebas.

### Done

Trabajo completado y validado.

## Organización

Cada funcionalidad principal del MVP se representa mediante una tarjeta independiente.

Cada tarjeta contiene:

* Objetivo funcional.
* Subtareas técnicas.
* Checklist de implementación.
* Criterios de finalización.

## Flujo de trabajo

1. Seleccionar tarea desde Todo.
2. Mover a In Progress.
3. Implementar y probar.
4. Mover a Review.
5. Validar funcionamiento.
6. Mover a Done.
